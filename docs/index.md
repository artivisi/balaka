# Dokumentasi Balaka

Balaka adalah aplikasi akuntansi untuk usaha kecil dan menengah di Indonesia. Fitur lengkap: pembukuan double-entry, kepatuhan pajak (PPN, PPh 21/23/Badan, e-Faktur, Coretax), payroll + BPJS, inventori & produksi, aset tetap, faktur & tagihan, rekonsiliasi bank, REST API.

## Pilih Panduan Sesuai Kebutuhan Anda

### 📘 Tutorial Akuntansi
**Untuk: Pemilik usaha, staf keuangan**

Panduan langkah-demi-langkah untuk mencatat keuangan usaha Anda. Tidak perlu latar belakang akuntansi.

- [Panduan Umum](tutorial-umum.html) — konsep dasar akuntansi, debit/kredit, template jurnal, PPN, PPh, BPJS, payroll, aset tetap, tutup buku
- [IT Service](tutorial-it.html) — jasa konsultasi IT (PKP, PPN, PPh 23)
- [Online Seller](tutorial-seller.html) — penjualan marketplace (UMKM, PPh Final)
- [Coffee Shop](tutorial-coffee.html) — kedai kopi dengan produksi (BOM, COGS)
- [Campus](tutorial-campus.html) — perguruan tinggi (SPP, beasiswa, hibah)
